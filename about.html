Our Technology

Methodology

How We Detect Counterfeit Notes

Combining contrastive learning with visual similarity analysis for accurate detection.

Contrastive Learning

Our system uses SimCLR (a contrastive learning framework) to learn meaningful representations of banknotes without requiring labeled counterfeit examples during training.

Visual Embeddings

Each banknote is converted into a 128-dimensional vector that captures its visual features. Genuine notes cluster together in this embedding space.

Anomaly Detection

New notes are compared against the genuine cluster. Those falling outside a similarity threshold are flagged as potential counterfeits.

Technical Architecture

Our system is built with robust, scalable technologies to ensure reliable performance.

Frontend

Responsive web interface built with HTML, CSS, and JavaScript. Optimized for both desktop and mobile devices.

Backend

FastAPI Python server handling image processing and model inference. Efficiently processes requests with async capabilities.

Machine Learning

PyTorch implementation of SimCLR trained on thousands of genuine Naira notes. OpenCV used for image preprocessing.

System Flow

1

User uploads image of Naira note through web interface

2

Image is preprocessed (resized, normalized) using OpenCV

3

Pretrained SimCLR model generates visual embedding

4

Embedding compared to genuine note cluster using cosine similarity

5

Results returned to user with visual explanations
